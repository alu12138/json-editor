# 粘贴 JSON 功能说明

## 功能介绍

在上传 JSON 文件按钮的旁边添加了一个"**粘贴 JSON**"按钮，方便用户直接粘贴或输入 JSON 内容，无需本地文件。

## 使用方式

### 方式 1：直接粘贴

1. 点击"粘贴 JSON"按钮
2. 弹窗打开，显示一个大的文本输入框
3. **Ctrl+V** (Windows) 或 **Cmd+V** (Mac) 粘贴 JSON 内容
4. 点击"确定"按钮
5. JSON 加载成功，效果等同于上传文件

### 方式 2：直接输入

1. 点击"粘贴 JSON"按钮
2. 在文本框中直接输入 JSON 内容
3. 点击"确定"按钮
4. JSON 加载成功

### 方式 3：复制粘贴

1. 从其他应用或网页复制 JSON 文本
2. 点击"粘贴 JSON"按钮
3. 在弹窗中粘贴
4. 点击"确定"按钮

## 弹窗特性

✅ **支持滚动**
- 文本框内容过多时会自动显示滚动条
- 支持键盘快捷键（Ctrl+A 全选、Ctrl+C 复制等）

✅ **大文本输入**
- 支持粘贴大量 JSON 内容
- 默认显示 12 行高度，超出时滚动

✅ **提示信息**
- 弹窗顶部有友好的提示文字
- 占位符提示"在此粘贴 JSON 内容..."

✅ **错误提示**
- 如果 JSON 格式不正确，会弹出错误提示
- 错误消息会显示具体的问题位置

## 界面布局

```
左侧编辑区顶部：
┌─────────────────────────────────────┐
│ [上传 JSON 文件] [粘贴 JSON]        │
└─────────────────────────────────────┘
```

## 常见场景

### 场景 1：从在线 JSON 编辑器复制

1. 在 jsoncrack.com 或类似网站创建 JSON
2. 复制全部内容
3. 打开本 JSON 编辑器，点"粘贴 JSON"
4. Ctrl+V 粘贴，点"确定"

### 场景 2：从 API 响应复制

1. 从 API 文档或 Postman 中复制响应 JSON
2. 点"粘贴 JSON"
3. 粘贴内容，点"确定"
4. 开始编辑

### 场景 3：手动输入小型 JSON

1. 点"粘贴 JSON"
2. 直接在文本框中输入 JSON
3. 点"确定"

## 加载后的效果

粘贴 JSON 成功后：
- ✅ 左侧显示树形菜单结构
- ✅ 右侧显示实时预览
- ✅ 支持所有编辑功能（修改、添加、删除等）
- ✅ 默认勾选"只显示修改"
- ✅ 支持搜索、批量修改等功能

## 常见问题

**Q: 如果 JSON 格式错误会怎样？**
A: 会显示"JSON 格式错误：具体错误信息"的提示，需要修改 JSON 后重新尝试。

**Q: 可以粘贴多少内容？**
A: 理论上没有限制，但建议不超过 50MB（性能考虑）。

**Q: 粘贴的 JSON 会自动排序吗？**
A: 不会，保持原有的顺序。加载后显示的树形菜单顺序与原 JSON 一致。

**Q: 和上传文件的区别是什么？**
A: 功能完全相同，只是数据获取方式不同。一个从文件读取，一个从粘贴内容读取。

**Q: 可以同时粘贴多个 JSON 吗？**
A: 不可以，每次只能粘贴一个 JSON。如果需要多个，可以逐个加载。
