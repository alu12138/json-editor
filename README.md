# JSON 编辑器

一个轻量级的 JSON 可视化编辑工具，完全由AI编程，支持树形展示、搜索、编辑、批量修改等功能。

## 项目背景
- 主要应用场景为运营需要修改 JSON 配置，手动修改效率低且容易出错。
## 功能特性

- ✅ 上传 JSON 文件
- ✅ 树形结构可视化
- ✅ 搜索过滤
- ✅ 编辑/删除/新增字段
- ✅ 批量修改匹配字段
- ✅ 预览 JSON
- ✅ 一键复制
- ✅ 导出 JSON 文件

## 技术栈

- React 18
- Ant Design 5
- Vite

## 本地开发

```bash
npm install
npm start
```

## 构建部署

```bash
npm run build
```

生成的 `dist` 目录即可部署到任何静态托管服务。

详见 [DEPLOY.md](./DEPLOY.md)

无需安装任何环境，直接访问部署后的链接即可使用。
