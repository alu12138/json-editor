# JSON 编辑器

一个高性能的 JSON 可视化编辑工具，完全由AI编程，支持树形展示、搜索、编辑、批量修改等功能。

## 项目背景

- 主要应用场景为运营需要修改 JSON 配置，手动修改效率低且容易出错。
- 支持超大文件（176KB+ 测试成功，加载时间 < 500ms）

## 功能特性

### 核心功能
- ✅ 上传 JSON 文件
- ✅ 粘贴 JSON 内容（无需上传文件）
- ✅ 树形结构可视化，显示 key: value
- ✅ 实时预览，修改部分用红色标记
- ✅ 搜索和查找功能

### 编辑功能
- ✅ 编辑叶子节点的值
- ✅ 为数组和对象添加新项
- ✅ 删除任意节点
- ✅ 批量修改匹配项目（支持复选框选择）
- ✅ 导出 JSON 文件
- ✅ 一键复制 JSON

### 预览功能
- ✅ 实时预览 JSON
- ✅ 搜索预览内容，高亮显示
- ✅ 只显示修改（默认勾选）
- ✅ 修改统计（显示已修改 N 处）

## 快速开始

### 方式 1：上传文件
1. 点击"上传 JSON 文件"
2. 选择本地 JSON 文件
3. 等待加载完成

### 方式 2：粘贴 JSON
1. 点击"粘贴 JSON"
2. 粘贴或输入 JSON 内容
3. 点击"确定"加载

### 编辑 JSON
1. 点击树形节点的"编辑"按钮修改值
2. 点击"添加"按钮为数组/对象添加项
3. 修改自动在预览区用红色标记

### 搜索功能
1. 输入搜索词
2. 点击"查找"按钮
3. 树形菜单过滤显示匹配项

### 批量修改
1. 输入搜索词并查找
2. 点击"批量修改"
3. 选择或取消选择要修改的项
4. 输入新值，点击"确定"
